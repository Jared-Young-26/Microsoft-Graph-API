# Settings

## Configuration

Required fields:

- Tenant ID
- Client ID
- Client Secret

Optional fields:

- Graph user ID
- OneDrive drive ID
- SharePoint admin URL

## Keychain integration

Use the **Keychain** toggle to store secrets securely on supported platforms.

## Environment lock

Enable **Lock** to prevent accidental changes to configuration.

> [!NOTE]
> After updating config, always click **Reload config** to apply changes.
