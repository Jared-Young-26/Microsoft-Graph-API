# Baselines

Baselines let you compare current state to a known-good snapshot.

## Golden baselines

1. Capture a snapshot of a healthy subject.
2. Mark it as **Golden**.
3. Compare current snapshots to the golden baseline.

Use cases:
- Detect drift on critical servers
- Confirm that changes are intentional
- Prove what changed since last week

## Compare to golden

When comparing:

- Review the **Triage** summary
- Focus on security or configuration drift
- Export an incident bundle if needed
