# Governance

## Project Stewardship

Graph Admin Studio was created and is maintained by Jared Young. The maintainer has final authority over the roadmap, architecture, merges, and releases.

## Decision Making

Feedback and discussion are welcome. Consensus is encouraged but not required. Final decisions rest with the maintainer to preserve clarity, quality, and architectural integrity.

## Contributions

Contributions are welcome via pull requests. Acceptance is not guaranteed. The maintainer may request changes or reject contributions for any reason, including scope, safety, or design alignment.

## Forks

Forks are permitted under the license. The canonical project lives in the primary repository maintained by Jared Young.

## Code of Conduct

Respectful, professional interaction is required in all project spaces.
